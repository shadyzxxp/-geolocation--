# Требования к проекту «Навигатор по ТЦ»

## Функциональные требования

1. Поиск по магазинам и услугам
   - Поиск по названию магазина
   - Фильтр по категории (одежда, еда, развлечения, услуги и т.д.)

2. Интерактивная карта ТЦ
   - Отображение этажей
   - Выделение выбранного магазина на карте

3. Построение маршрута
   - Выбор точки старта (вход, текущая точка, магазин)
   - Построение оптимального маршрута до выбранного магазина

4. Акции и мероприятия
   - Список текущих акций
   - Список ближайших мероприятий в ТЦ

## Нефункциональные требования

- Web-приложение, адаптивное под мобильные устройства
- Понятный и минималистичный интерфейс
- Возможность дальнейшего расширения (добавление новых ТЦ, интеграция с реальной геолокацией)
