Навигатор по ТЦ

Проект:Создание geolocation-сервиса с умной навигацией

Цель: разработать веб-приложение, которое помогает посетителям торгового центра:
- находить магазины и услуги;
- строить маршрут внутри ТЦ;
- просматривать актуальные акции и мероприятия.

Ключевые функции

- Интерактивная карта торгового центра с поиском
- Построение маршрутов до магазинов / услуг
- Отображение акций и мероприятий

 Аналоги

- Mall Maps  
- Яндекс.Карты (режим ТЦ)

 Технологический стек (план)

- Frontend: React, Vite, Leaflet
- Backend: Node.js, Express
- DB: MongoDB
