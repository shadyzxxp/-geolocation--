# -geolocation--
• Навигатор по ТЦ : Помогает посетителям находить магазины и услуги.


Суть проекта:
Веб-приложение для навигации внутри торгового центра: поиск магазинов, построение маршрутов, просмотр акций и мероприятий.

Основные функции:

 Интерактивная карта ТЦ
 Поиск магазинов и услуг
 Маршрутизация внутри здания
 Акции и события

Проблема:
Посетители часто теряются в больших ТЦ и не знают, где находятся нужные магазины или какие акции сейчас проходят.

Наше решение:
Простой веб-сервис с картой, умным поиском и маршрутом от точки А до магазина.

Аналоги: Mall Maps, Яндекс.Карты (режим ТЦ)
